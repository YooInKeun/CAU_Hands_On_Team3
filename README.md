중앙대학교 컴퓨터공학과 커뮤니티 웹 사이트
=

스터디, 팀플 매칭 및 강의 후기 공유 서비스
-

## 서비스 기능

```
- 컴퓨터공학과 강의 조회

- 스터디, 팀플, 강의 후기 게시글 및 댓글 작성

- 원하는 유저에게 쪽지 보내기
```
<br>

## 기술 스택

```
- Django 2.1.1

- Sqlite3

- Cloud: AWS ElasticBeanstalk
```
